Vendor Management System

Welcome to the Vendor Management System

This system allows you to manage vendor profiles, track purchase orders, and evaluate vendor performance metrics.

Here are some core features of the system:

  • Manage vendor profiles: Create, read, update, and delete vendors.
  • Track purchase orders: Manage purchase orders including creation, updating, and deletion.
  • Evaluate vendor performance: Calculate metrics like on-time delivery rate, quality rating, response time, and fulfillment rate.

You can also find the source code on GitHub.
